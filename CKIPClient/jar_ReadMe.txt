使用方法
==============
需求: 
需安裝Java runtime enviroment 1.5以上版本。
安裝及環境設定請參考java程式語言的相關教學。
------------------------------------------------
執行:
執行方法與exe檔類似，僅原命令列
CkipClient.exe [prop] [input dir] [output dir]
改成:
java -jar CkipClient.jar [prop] [input dir] [output dir]
=============================================================
demo：
run.bat, run.sh是執行範例，測試前請先修改ckipsocket.propeties的帳號資訊
